创建博客（hexo）

发表于2025-07-29|更新于2025-10-11|开发编程
|浏览量:

需要的物品

  1. 一台服务器(阿里云、腾讯云、华为云等)
  2. Liunx系统（Ubuntu）

可选项

  1. 域名
  2. 备案
  3. SSL

具体步骤：

1. 安装 fnm

2. 解压

3. 移动到 ~/.local/bin

4. 将fnm加入环境变量

5. 安装 nodejs

6. 安装 hexo

7. 初始化 hexo

测试搭建是否完成,使用命令 hexo server
访问 http://【服务器IP】:4000 即可看到 hexo 博客

8. 安装Butterfly主题

点击进入 Butterfly官方教程
在你的 Hexo 根目录内

然后将主题配置文件复制到 Hexo 根目录

升级方法：在主题目录下，运行 git pull

应用主题

修改 Hexo 根目录下的_config.yml ，把主题改为 butterfly

如果你遇到了导航栏布局问题，请修改themes\butterfly\source\css\_layout\head.styl加一个display: flex;
